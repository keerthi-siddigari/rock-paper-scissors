# rock-paper-scissors
A fun and interactive Rock,Paper,Scissors game built using HTML,CSS and Javascript.

#Features
-Play with the Computer
-Beginner friendly,easy to play
-Keeps track of the scores
-Responsive

#How to Play
-Select your move fromyhe listed choices(rock,paper,scissors)
-Computer selects a move
-The result is displayed instantly (It's a tie,You Win,You Lose)

#Suitable for all devices such as
-Mobile Phones
-Tablets
-Laptops
-Large screens
